
# Лабораторная работа №2

**Тема:** Регулярные выражения и Unit-тестирование в Python
**Вариант:** 16. Проверка и поиск синтаксически корректного времени в формате ЧЧ:ММ:СС

## Цель работы

Изучить и применить регулярные выражения в языке Python для поиска и проверки корректности временных значений в формате **ЧЧ:ММ:СС**, а также освоить основы **unit-тестирования** для проверки корректности работы программы.

## Используемые технологии

* **Язык:** Python 3.13
* **Библиотеки:**

  * `re` — для работы с регулярными выражениями
  * `requests` — для выполнения HTTP-запросов
  * `unittest` — для unit-тестирования

## Установка и запуск

1. Убедитесь, что установлен Python версии **3.10+**
2. Установите недостающие зависимости:

   ```bash
   pip install requests
   ```
3. Запустите программу:

   ```bash
   python main.py
   ```
4. Следуйте инструкциям в консоли:

   ```
   1 — Проверить введённое время
   2 — Найти корректное время в файле
   3 — Найти корректное время на веб-странице
   ```

## Запуск тестов

Для проверки корректности работы программы выполните:

```bash
python -m unittest discover
```
